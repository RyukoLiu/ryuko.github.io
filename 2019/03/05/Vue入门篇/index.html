Vue入门篇 - 开发去哪儿网App中涉及的命令

作者:Naiaky    发布于: 2019-03-05

把线上代码克隆到本地
git clone git@gitee.com:Naiaky/QUNR.git

✿

安装vue——cli
$ npm install –global vue-cli

✿

创建基于webpack打包工具的项目
vue init webpack QUNAR

✿

浏览本地仓库
git status

✿

将本地文件添加到git的缓冲区
git add .

✿

将本地代码推到线上
git commit -m ‘description’
git push

✿

解决移动端300毫秒延迟点击问题
npm install fastclick –save

✿

npm install stylus –save
npm install stylus-loader –save

✿

轮播图
npm install vue-awesome-swiper@2.6.7 –save

✿

发送ajax（axios）
npm install axios –save

✿

实现滑动小细节
npm install better-scroll –save

✿

npm install vuex –save

✿

解决一些手机浏览器兼容性（不支持promise）问题
npm install babel-polyfill –save

✿

vue的脚手架会将代码打包编译，生成浏览器可以运行的代码（压缩后的）
（生成dist目录，把目录下代码移动到后端根目录下就完成了最简单的项目上线）
npm run build

✿

代码地址： https://gitee.com/Naiaky/Travel
